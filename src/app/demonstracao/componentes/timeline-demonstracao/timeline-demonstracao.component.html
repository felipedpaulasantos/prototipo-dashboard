Timeline

O componente Timeline apresenta uma linha do tempo simples com ícones e cores que indicam a natureza de cada acontecimento.

O atributo items deve sempre receber uma lista de TimelineItem.

O componente recebe a lista de TimelineItem e desenha, por padrão na vertical, a linha do tempo.

Os estados possíveis podem ser informados utilizando a enum TimelineState, ou informando a string diretamente (ex. "success").

A data deve ser informada pelo atributo date, do tipo Date, ou pelo atributo dateString, do tipo String.

Caso a formatação padrão do objeto Date não seja a desejada, pode-se informar pelo atributo dateFormat qualquer um dos formatos suportados pelo DatePipe do Angular: https://angular.io/api/common/DatePipe .


Caso a lista de acontecimentos seja extensa, é possível atribuir valores para os propriedades do componente, como height ou width, pelo atributo styles.

O componente exibirá uma barra de rolagem e destacará a parte final da lista por padrão
