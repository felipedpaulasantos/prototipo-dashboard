Select

A diretiva selectCaixa em conjunto com o componente cx-select permite a estilização e validação automática dos selects.
O componente utiliza a biblioteca Bootstrap-Select .

A validação observa os Validators do Angular.

Todas as opções da biblioteca Bootstrap-Select estão disponíveis.

Para opções carregadas de forma assíncrona, é necessário atualizar a propriedade initTrigger para recarregar o componente.
