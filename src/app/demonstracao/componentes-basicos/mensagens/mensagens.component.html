As notificações que não demandam interação do usuário devem ser apresentadas no formato "toast".

A chamada é feita pelo Toastr Service, da biblioteca Ngx-Toastr.

       

Quando houver a necessidade de interação do usuário com o conteúdo apresentado, deve-se utilizar um modal.

Pode-se exibir pelo serviço ModalService, com várias opções de customização.

O "spinner" de carregamento pode ser chamado pelo NgxSpinnerService, da biblioteca Ngx Spinner.
Notificações (Toastr)

Biblioteca Ngx-Toastr

No módulo: 
import {{ '{' }} ToastrModule  {{ '}' }} from 'ngx-toastr';
No componente: 
import {{ '{' }} ToastrService {{ '}' }} from 'ngx-toastr';

Spinner (NgxSpinner)

Biblioteca NgxSpinner

No AppModule: 
import {{ '{' }} NgxSpinnerModule  {{ '}' }} from 'ngx-toastr';
No AppComponent (.ts): 
import {{ '{' }} NgxSpinnerService {{ '}' }} from 'ngx-spinner';
No AppComponent (.html): 
<cx-spinner [fullscreen]=true></cx-spinner>

Modal
No AppComponent (.html): 
<cx-modal></cx-modal>
No componente: 
import {{ '{' }} ModalService {{ '}' }} from 'src/app/guia-caixa/services/modal.service';
