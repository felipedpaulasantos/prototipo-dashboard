Forms

Os estilos dos campos de formulário seguem o padrão definido pelo Bootstrap.

As variantes de tamanho, status e layout descritas na documentação do link acima devem ser seguidos sem alteração.
  • Visão Geral
  • Exemplos

Para campos do tipo "input" e "text-area" utilizar a classe padrão "form-control" do Bootstrap.

O espaçamento entre campos pode ser implementado de duas maneiras:
- Envolvendo o label e o campo num container com a classe "form-group";
- Utilizando a classe auxiliar do Bootstrap "mb-3" (espaçamento inferior de 1rem).

Para campos do tipo "select" utilizar por padrão o componente da biblioteca Ng-Select, que já oferece funcionalidades de filtro, autocomplete e multi-seleção.

Caso não seja possível inserir o componente, utilizar a classe "form-control" no elemento "select".

O Checkbox deve seguir o estilo "custom".

O Radio, assim como o checkbox, também deve seguir o estilo "custom".

O componente 'cx-button-control' se comporta como radio ou checkbox, conforme o valor passado no atributo 'type'.

Para o elemento Range utilizar a classe "custom-range".

As classes do Angular Forms (ng-dirty, ng-valid, ng-invalid) alteram o visual dos campos conforme seu estado de validação.
