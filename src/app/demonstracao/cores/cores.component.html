Cores

  • Cores temáticas
  • Cores constantes

As cores temáticas, seguindo o padrão do Bootstrap, representam conceitos e podem variar conforme a mudança do tema global da aplicação.

As variáveis das cores temáticas devem ser utilizadas em todos os elementos e componentes da aplicação.

As classes do Bootstrap de cor de fundo (bg-nome_do_tema) e cor de texto (text-nome_do_tema) estão disponíveis para todos os temas.

Para utilizar uma variável CSS, deve-se apenas informar seu valor seguindo a sintaxe abaixo, em qualquer lugar da aplicação:

CSS
Nome Variável Valor Demonstração Utilização
{{ cor.nome }} {{ cor.variavel }} {{ cor.valor }}

Todas as cores definidas pelo Design System Caixa estão presentes aqui.

Estas cores são imutáveis e devem ser usadas apenas como referência para as variáveis temáticas.
